## Project Description
A simulated pipelined processor implemented in C. The processor features:
- Von Neumann architecture with 2048x32 memory
- 32 registers including a zero register (R0)
- 12 instructions across R, I, and J formats
- 5-stage pipeline: IF, ID, EX, MEM, WB
